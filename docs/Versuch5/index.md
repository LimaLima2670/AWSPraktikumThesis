# Versuch 5  
## Steuerung über CLI und SDK

In diesem Versuch wird untersucht, wie AWS-Ressourcen über Kommandozeilenwerkzeuge und Programmierschnittstellen gesteuert werden können.

Dabei wird insbesondere die Nutzung der AWS CloudShell, der AWS CLI sowie des Python SDK (boto3) betrachtet.

---

## Inhalt des Versuchs

Im Rahmen dieses Versuchs werden folgende Schritte durchgeführt:

- Einführung in die AWS CloudShell  
- Nutzung der AWS CLI zur Steuerung von Ressourcen  
- Programmatische Steuerung über das Python SDK (boto3)  
- Analyse der Ergebnisse
