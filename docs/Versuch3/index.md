# Versuch 3  
## Automatisierter Serverless-Workflow

In diesem Versuch wird ein automatisierter Workflow zur Videoverarbeitung in der AWS Cloud aufgebaut.

Dabei werden mehrere AWS-Dienste miteinander kombiniert, um einen vollständig automatisierten Videoverarbeitungsprozess zu realisieren.

---

## Inhalt des Versuchs

Im Rahmen dieses Versuchs werden folgende Schritte durchgeführt:

- Vorbereitung der S3-Infrastruktur  
- Einrichtung von SNS für Benachrichtigungen  
- Erstellung von Lambda-Funktionen  
- Test des automatisierten Workflows  
- Analyse der Ergebnisse
