# Versuch 1  
## Grundlagen eines AWS-basierten Transcoding-Workflows

In diesem Versuch werden die grundlegenden AWS-Dienste vorgestellt, die für einen einfachen cloudbasierten Video-Transcoding-Workflow benötigt werden.

Dabei werden folgende Schritte durchgeführt:

- Einrichtung eines Amazon S3 Buckets  
- Upload eines Beispielvideos  
- Transcodierung über AWS MediaConvert  
- Download der erzeugten Ausgabedateien
