# Versuch 2  
## Content-Auslieferung über CDN

In diesem Versuch wird untersucht, wie transcodierte Inhalte über ein Content Delivery Network ausgeliefert werden können.

Dabei werden folgende Schritte durchgeführt:

- Konfiguration der Storage-Eigenschaften  
- Einrichtung der NetStorage-Struktur  
- Bereitstellung der Inhalte  
- Test der Auslieferung über CDN
